# Exercício Java - Membros Estáticos/POO.

## Projeto de estudo em Java para praticar conceitos básicos de membros estáticos.

Esse projeto foi criado como solução para o execício sobre membros estáticos do tópico Introdução à Programação Orientada a Objetos do curso de Java do Nélio Alves (Udemy). Os objetivos do exercício eram:
1. Fazer um programa para ler a cotação do dólar e uma quantia em dólar para ser comprado.
2. Informar qual será o valor pago (em reais), considerando uma taxa de 6% sobre o valor do dólar no valor final.
